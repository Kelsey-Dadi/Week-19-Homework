Hello, I am Kelsey Dadi!

Jr. UX/UI researcher and designer passionate about designing solutions that truly meet the needs of users.

My unique background working at Target Cooperation in Human Resources for configuration, database management, and UI and Micro-service development, brought me to UI/UX design.

My specialties are communicating with users, designers, and developers to come up with the best possible solution. Using problem solving and critical thinking to work as a team to solve any problem along the way.

My UX Skills

Research: A/B Testing, Affinity Diagraming, Card Sorting, Persona Creation, User Interviews/Research, Usability Testing

Design: Using Figma to create wireframe prototypes of low, mid, or high fidelity getting user feedback early and often.

Front End Development: HTML, CSS, Java React, Bootstrap, and Github

My Work
